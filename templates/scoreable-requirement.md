# Scoreable Requirement Template

Use this template only where measurable supplier differentiation creates procurement value.

```yaml
id:
title:
capability:
type: scoreable
evaluation:
  method: score
  scale: 1-4
  justification:
requirement:
  text:
scoring:
  4:
  3:
  2:
  1:
evidence:
  - 
verification:
  method:
notes:
```

## Requirements for scoring

A scoreable requirement shall:

1. Have mutually exclusive scoring levels.
2. Have objectively verifiable criteria.
3. Justify evaluation effort.
4. Not duplicate mandatory baseline compliance.
5. Fit within the configured maximum scoreable ratio.
