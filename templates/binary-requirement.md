# Binary Requirement Template

Use this template for mandatory SHALL requirements evaluated as Yes/No.

```yaml
id:
title:
capability:
type: mandatory
evaluation:
  method: binary
  scale:
    yes: Requirement is fulfilled.
    no: Requirement is not fulfilled.
requirement:
  text:
evidence:
  - 
verification:
  method:
notes:
```

## Markdown output

### Requirement

`<single atomic SHALL statement>`

### Evaluation

Yes / No

### Evidence

`<short evidence description>`

### Verifier notes

`<optional evaluator guidance>`
