# Authorization Scoring Guidance

Default position:

> Binary mandatory requirement.

Scoring may be considered when:

1. Suppliers can exceed the mandatory baseline.
2. The difference is objectively measurable.
3. The difference improves supplier selection.
4. Evaluation effort is justified.

Scoreable variant:

> `access_control_maturity`

Score access control maturity where fine-grained permissions, delegated administration, or segregation controls create value.
