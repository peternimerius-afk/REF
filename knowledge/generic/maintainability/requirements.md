# Maintainability Requirement Patterns

## Mandatory baseline

> The supplier shall provide a documented maintenance process.

Evaluation:

> Yes / No

Evidence:

> Maintenance process

## When to split

Split this capability when the requirement also includes:

- Evidence requests
- Reporting obligations
- Testing obligations
- Support obligations
- Separate technical mechanisms
- Contractual remedies

## Scoreable candidate

Scoreable variant: `lifecycle_maturity`

Use only if higher maturity creates measurable procurement value.
