# Backup Requirement Patterns

## Mandatory baseline

> Backup shall be performed at least daily.

Evaluation:

> Yes / No

Evidence:

> Backup policy

## When to split

Split this capability when the requirement also includes:

- Evidence requests
- Reporting obligations
- Testing obligations
- Support obligations
- Separate technical mechanisms
- Contractual remedies

## Scoreable candidate

Scoreable variant: `backup_maturity`

Use only if higher maturity creates measurable procurement value.
