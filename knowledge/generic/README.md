# Generic NFR Knowledge Pack

**Version:** 0.1.0  
**Status:** Draft

The Generic NFR Knowledge Pack contains capabilities that are commonly relevant across IT and digital service procurements.

This pack is domain-neutral. It applies before aviation, SaaS, AI, AODB, BRS, RMS, FIDS, or other domain-specific packs are loaded.
