# Generic NFR Knowledge Pack

This pack contains technology- and industry-neutral non-functional requirement knowledge.

It should be loaded for most procurement projects.

## Capability groups

- Availability and continuity.
- Security and access.
- Logging and monitoring.
- Backup and recovery.
- API and integration.
- Support and maintainability.
- Performance and capacity.

## Engineering guidance

Generic NFRs are normally mandatory binary requirements.

Scoreable requirements should be reserved for maturity or quality dimensions where supplier differentiation matters.
