# Configuration Management Requirement Patterns

## Mandatory baseline

> System configuration changes shall be traceable.

Evaluation:

> Yes / No

Evidence:

> Configuration management process

## When to split

Split this capability when the requirement also includes:

- Evidence requests
- Reporting obligations
- Testing obligations
- Support obligations
- Separate technical mechanisms
- Contractual remedies

## Scoreable candidate

Scoreable variant: `operations_maturity`

Use only if higher maturity creates measurable procurement value.
